# KI-Prompts — Sammlung von Vorlagen

Kurzbeschreibung
- Diese Sammlung enthält wiederverwendbare KI-Prompts zur Unterstützung von Datenanalyse-, KNIME- und Reporting-Workflows.

Enthaltene Prompts (Auswahl)
- **KNIME Full-Stack & PowerBI**: Leitfaden für kompletten KNIME-Workflow und PowerBI-Integration — [Prompt_002_KNIME_Full-Stack-Analytics-Architect-&-PowerBI-Integration-Specialist.md](src/019_KI_Prompts/Prompt_002_KNIME_Full-Stack-Analytics-Architect-&-PowerBI-Integration-Specialist.md)
- **Reporting & Dashboards**: Prompt für Erstellung von Reports, Automatisierung und Dashboard-Design (im selben KNIME-Prompt ergänzt)
- **Präsentation der Ergebnisse**: Prompt zur Erstellung von Präsentationen (PowerPoint/PDF) basierend auf Analyseergebnissen
- **Kanban-Board Analyse**: Prompt zur Projektanalyse (Kumulatives Flussdiagramm, Burnup/Burndown, Durchlauf-/Zykluszeiten, Aufgaben- und Nutzerverteilung)

Verwendung
- Prompt öffnen, Platzhalter [in eckigen Klammern] mit Projektdetails ersetzen.
- Kopieren Sie den angepassten Prompt in Ihr Chat-Interface (z. B. ChatGPT, Claude) oder in ein Automatisierungs-Tool.

Best Practices
- Beschreiben Sie Datenquelle(n), Beispiel-Daten und Ziel klar (Zielgruppe, Metriken, Zeiträume).
- Fügen Sie Beispiele/Tabellen mit 5–10 Zeilen hinzu, damit der Prompt präzise Antworten liefern kann.

Beispiel
1. Öffnen Sie die Datei mit dem KNIME-Prompt:

	`src/019_KI_Prompts/Prompt_002_KNIME_Full-Stack-Analytics-Architect-&-PowerBI-Integration-Specialist.md`

2. Ersetzen Sie: `[DEIN ZIEL]`, `[KURZE DATENBESCHREIBUNG]`, etc.

Contribution
